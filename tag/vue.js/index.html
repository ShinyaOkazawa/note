disabled 属性は空でも truthy になる
Date: 4/25/2024Category: Front-EndTag: Vue.js

disabled 属性は空の場合も truthy になるので注意。 基本的には boolean によって disabled 属性の付け外しを行うのが紛らわしくなくて良い。

<input type="text" disabled="">
同名省略記法
Date: 4/24/2024Category: Front-EndTag: Vue.js

バインドする値と属性が同じ名前を持っている場合、下記のように省略した記法ができる。 Vue3.4以上から使用できる。

小さいコンポーネントであれば使う機会がありそう。

<script setup lang="ts">
const id = 'hoge'
</script>
<template>
  <div :id>hoge</div>
</template>
