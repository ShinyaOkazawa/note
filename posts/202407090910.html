Slack通知を試してみた

GitHub のリポジトリに push したら Slack に通知する仕組みを作ってみた。

方法

  • Slack で Incoming Webhooks の設定をする
  • Webhook URL を GitHub の対象リポジトリの Settings にて Secrets に設定する
  • Secrets に設定した定数を用いて .github/workflows/slack-notify.yml を作成する
