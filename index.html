8/11/2024: import のファイルパス補完には tsconfig の設定が必要
Category: Front-EndTag: Vite

Vite 環境、VSCode でコーディングするときに、パスの補完がされないと自力で相対パスを書かないといけないので面倒すぎる。

必要な設定は2つ。

vite 設定 の resolve.alias に解決したいパスを指定。

export default defineConfig({
  plugins: [vue()],
  resolve: {
    alias: {
      '@': fileURLToPath(new URL('./src', import.meta.url))
    }
  }
})

7/9/2024: Slack通知を試してみた
Category: miscTag: Slack

4/24/2024: 同名省略記法
Category: Front-EndTag: Vue.js

バインドする値と属性が同じ名前を持っている場合、下記のように省略した記法ができる。 Vue3.4以上から使用できる。

小さいコンポーネントであれば使う機会がありそう。

<script setup lang="ts">
const id = 'hoge'
</script>
<template>
  <div :id>hoge</div>
</template>
