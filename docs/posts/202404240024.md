バインドする値と属性が同じ名前を持っている場合、下記のように省略した記法ができる。
Vue3.4以上から使用できる。

小さいコンポーネントであれば使う機会がありそう。

```vue
<script setup lang="ts">
const id = 'hoge'
</script>
<template>
  <div :id>hoge</div>
</template>
```

[同名省略記法](https://ja.vuejs.org/guide/essentials/template-syntax.html#same-name-shorthand)と呼ぶらしい。
